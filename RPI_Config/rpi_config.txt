## Setting up RPI Config

1. Install the Raspi imager tool by running the command `sudo apt install rpi-imager`.
2. Launch the Raspi Imager application.
3. Click on "Choose OS".
4. Select "Raspberry Pi OS (other)".
5. Choose "Raspberry Pi OS Lite (32-bit)" without the desktop environment.
6. Click on "Choose storage" and select the SD card.
7. Click on the gear icon and enable SSH with password authentication.
8. SSH into the Pi by finding its IP address on the router dashboard.
9. Upload the repository to the Pi.
10. Install pip3 by running the command `sudo apt-get install python3-pip`.
11. Install i2c-tools by running the command `sudo apt-get install i2c-tools`.

## Light Sensor Installation

1. Install the `adafruit-circuitpython-si1145` library by running the command `sudo pip3 install adafruit-circuitpython-si1145`.
2. Run an I2C check to see if both sensors are registered by executing the command `i2cdetect -y 1`.
